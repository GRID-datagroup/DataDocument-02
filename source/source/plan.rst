观测计划文件
=============

  ``plan.txt``\ 是观测计划文件，文件构成如下：

+-----------------+------------------+--------------------------+
|       字段      |       含义       |           说明           |
+-----------------+------------------+--------------------------+
|       Time      |       时间       | 20 Apr 2021 00:00:00.000 |
+-----------------+------------------+--------------------------+
|  Detic Latitude | 探测器星下点纬度 |                          |
+-----------------+------------------+--------------------------+
| Detic Longitude | 探测器星下点经度 |                          |
+-----------------+------------------+--------------------------+
|  Detic Altitude |    探测器高度    |                          |
+-----------------+------------------+--------------------------+
|  RightAscension |                  |                          |
+-----------------+------------------+--------------------------+
|   Declination   |                  |                          |
+-----------------+------------------+--------------------------+
|        x        |                                             |
+-----------------+                                             |
|        y        |               WGS84坐标系坐标               |
+-----------------+                                             |
|        z        |                                             |
+-----------------+---------------------------------------------+
|        x        |                                             |
+-----------------+                                             |
|        y        |            J2000坐标系坐标系坐标            |
+-----------------+                                             |
|        z        |                                             |
+-----------------+---------------------------------------------+
